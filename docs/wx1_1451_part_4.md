# Deploying your Customization

## 4.1 Setting and using tags to filter devices
Device Tags are one of the most underutilized features for sorting and filtering devices. 
Not only are tags useful in Control Hub, but they are also extremely beneficial when working with the Webex RESTful APIs, providing one of the most effective ways to filter devices.
